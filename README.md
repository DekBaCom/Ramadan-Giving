# Ramadan Donation Hub (สะพานบุญลดหย่อนภาษี)

เว็บไซต์รวบรวมรายชื่อมูลนิธิ สถานศึกษา โรงพยาบาล และมัสยิด ที่เปิดรับบริจาคในช่วงเดือนรอมฎอน พร้อมข้อมูลสิทธิประโยชน์ลดหย่อนภาษี (e-Donation) ตามกฎหมายกรมสรรพากรไทย และเครื่องมืออำนวยความสะดวกสำหรับมุสลิมแบบครบวงจร

## 🌟 ฟีเจอร์ใหม่และการอัปเดตล่าสุด
*   **เครื่องคำนวณซะกาต (Zakat Calculator)**: ระบบช่วยคำนวณซะกาตทรัพย์สินที่ 2.5% พร้อมระบบแจ้งเตือนเมื่อทรัพย์สินยังไม่ถึงพิกัดนิซอบ (Nisab)
*   **ข้อมูลอัซนาฟ 8 (Asnaf 8)**: ส่วนแสดงความรู้เกี่ยวกับบุคคล 8 จำพวกที่มีสิทธิ์รับซะกาตตามหลักการศาสนา
*   **เวลาละหมาดและนับถอยหลังละศีลอด (Prayer Times & Iftar Countdown)**: ดึงข้อมูลเวลาละหมาดแบบเรียลไทม์ตามตำแหน่งจริงของผู้ใช้ (Geolocation) พร้อมเวลานับถอยหลังสู่การละศีลอด (มัฆริบ)
*   **ระบบระบุพิกัดผู้ใช้ในแผนที่ (User Geolocation)**: แผนที่แบบ Interactive (Leaflet.js) ที่ตรวจจับและปักหมุดตำแหน่งปัจจุบันของผู้ใช้ และจัดเลื่อนกึ่งกลางแผนที่ให้อัตโนมัติ เพื่อค้นหาสถานที่บริจาครอบๆ ตัว
*   **แชร์บัตรบอกบุญ (Social Share Generator)**: สร้างรูปภาพบัตร QR Code พร้อมข้อมูลสถานที่บริจาค และแชร์ไปยัง Social Media (Facebook, LINE, Instagram, X) ได้ทันทีผ่าน Web Share API
*   **ระบบติดตามผลลัพธ์ (Impact Tracker)**: แถบหลอดความคืบหน้า (Progress Bar) สำหรับแต่ละแคมเปญบริจาคเพื่อแสดงความก้าวหน้าและกระตุ้นการมีส่วนร่วม
*   **บันทึกการบริจาคส่วนตัว (Personal Donation Log)**: ฟีเจอร์ช่วยจำและบันทึกสถานะ "บริจาคแล้ว" ของแต่ละมูลนิธิไว้ในเครื่อง (Local Storage) โดยไม่ต้องสมัครสมาชิก
*   **รองรับหลายภาษา (Multi-language)**: ปรับเปลี่ยนภาษาเป็น ภาษาไทย, English หรือ Arabic (รองรับการแสดงผลแบบ RTL) ได้ทันทีแบบไม่ต้องโหลดหน้าใหม่
*   **โหมดกลางคืน (Dark Mode)**: รองรับ Dark Mode เต็มรูปแบบ ปรับเปลี่ยนโทนสีและชุดแผนที่อัตโนมัติ พร้อมจำค่าไว้ในเครื่อง
*   **นับถอยหลังรอมฎอนปีถัดไป**: ระบบนับถอยหลังสู่วันเริ่มต้นเดือนรอมฎอน ปี 2027 (1448 AH) ที่หัวเว็บไซต์

## 📌 จุดเด่นของโครงการ
*   **รวบรวมสถานที่บริจาคที่หลากหลาย**: ครอบคลุมทั้งมัสยิด โรงเรียน และโรงพยาบาล
    *   **มัสยิด**: มัสยิดอับดุลลานุสรณ์, มัสยิดดารุ้ลฟาอีซีน, มัสยิดยามิอุ้ลมุตตะกีน, มัสยิดดารุลคอยรอต, มัสยิดนูรุสซาอาดะห์, มัสยิดสาเราะ และอื่นๆ
    *   **การศึกษา**: รร. พัฒนศาสน์วิทยา (ลดหย่อน 2 เท่า)
    *   **การแพทย์**: รพ.สุไหงโก-ลก, รพ.ปัตตานี, รพ.เบตง (ลดหย่อน 2 เท่า)
*   **ข้อมูลภาษีชัดเจน**: ระบุสิทธิลดหย่อนภาษี (1x หรือ 2x) ไว้อย่างชัดเจน
*   **ใช้งานง่าย (User Friendly)**:
    *   มีระบบ **QR Code Popup** สแกนผ่านโมบายแบงค์กิ้งเพื่อบริจาคได้ทันที
    *   มี **ปุ่มบันทึกรูป (Download)**: บันทึก QR Code ลงเครื่องเพื่อนำไปสแกนในแอปธนาคาร
    *   มี **ตัวกรอง (Filter)** เลือกประเภทการบริจาค เช่น การศึกษา, โรงพยาบาล หรือศาสนสถาน ได้ง่ายๆ
    *   ดีไซน์สวยงาม ทันสมัย รองรับการใช้งานบนมือถือ (Responsive Web Design)

## 🛠️ เทคโนโลยีและไลบรารีที่ใช้
*   **HTML5 / JavaScript (Vanilla)**: โครงสร้างและการจัดการลอจิกทั้งหมดแบบไม่ต้องพึ่งพา Framework หนักๆ
*   **Tailwind CSS**: ตกแต่งหน้าเว็บให้สวยงามด้วย Utility-first CSS Framework (ผ่าน CDN)
*   **Leaflet.js**: ระบบแผนที่ Interactive Map แบบ Open-source
*   **html2canvas**: ใช้ในการประมวลผลจับภาพหน้าจอ (DOM) เพื่อสร้างรูปบัตรบอกบุญ
*   **SweetAlert2**: ระบบ Popup/Modal สำหรับแจ้งเตือนผู้ใช้งานอย่างสวยงาม
*   **Lucide Icons**: ชุดไอคอนทันสมัยแบบ Open-source
*   **Aladhan API**: ดึงข้อมูลและคำนวณเวลาละหมาดทั่วโลกตามพิกัด
